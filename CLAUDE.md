## Agent skills

### Issue tracker

Issues live in GitHub Issues (`github.com/M-Kolacz/michalkolacz.com`). See
`docs/agents/issue-tracker.md`.

### Triage labels

Uses the five canonical defaults: `needs-triage`, `needs-info`,
`ready-for-agent`, `ready-for-human`, `wontfix`. See
`docs/agents/triage-labels.md`.

### Domain docs

Single-context layout — `CONTEXT.md` at the repo root and `docs/adr/`. See
`docs/agents/domain.md`.

### Testing

Four-tier strategy: unit, Storybook, E2E, and visual regression. See
`docs/agents/testing.md`.
